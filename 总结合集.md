2. string对象扩展一个方法删除前后空格

   ~~~javascript
   String.prototype.deletSpace = function(){
       let str = this
       while(str[0] === ' '){
           str = str.substring(1)
       }
       while(str[str.length -1] === ' '){
           str = str.substring(0,str.length -1)
       }
       return str
   }
   ~~~
